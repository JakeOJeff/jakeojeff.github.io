# two tools in one week - anim16 and a car physics library

## Date : 27-03-2023

okay so i went on a bit of a tool making spree this week. first i made anim16 (repo's called sensualmoves lmao dont ask) which is this little quickie animation tool for löve. making sprite animations by hand is SO tedious, youre sitting there tweaking frame timings forever, so i built something simple enough that literally anyone could use it. like so simple even grandma alice could edit it, thats the whole pitch

then like a day later i built seductivehood which is a 2d car physics library you can drop into your games. i wanted vehicle handling in löve, proper stuff, suspension and tire friction and steering that actually FEELS right. and thats the thing about car physics, its not about being mathematically correct, its about FEEL. i spent way too long just driving a little box around a blank screen tweaking numbers until it felt fun instead of floaty

the names for these are ridiculous i know. but i was having fun and thats kind of the point. both of these are meant for other game devs to just grab and use, which is becoming a theme for me, i really like building the tools more than i expected
