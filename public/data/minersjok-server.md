# started running my own minecraft server (minersjok)

## Date : 2020 (approx)

okay so i finally did the thing ive wanted to do forever, i started my OWN minecraft server. called it MinersJoK. its got minigames, survival, bedwars, the works. hosting it through aternos and slowly turning it into something real

spent all week setting up plugins. got land claim working so people cant just grief each others builds, added anti-grief stuff, tweaked permissions like a million times. i made it work for both cracked accounts (tlauncher and stuff) AND premium accounts because i want everyone to be able to join, not just people who bought the game. running a server is WAY more admin work than i expected lmao, its not just "turn it on", its constant babysitting and config files and people asking for stuff

but its so cool having a little world thats MINE that other people hang out in. this is the first time ive run something that other people actually depend on being up, and weirdly i kinda love that responsibility. its not code exactly but its the same feeling, you set up a system and then people live inside it
