# made the security thing official - i'm OCSP certified now

## Date : 09-2025

so all that pentesting i've been doing on the side, breaking into machines, poking at web apps, messing with android, i finally went and got it certified. i'm now an Offenso Certified Security Professional (OCSP), done through Offenso Hackers Academy right here in kochi. it covers both sides, offensive and defensive, which is exactly how i think about security anyway, you can't defend a thing properly until you understand how you'd attack it

this has been brewing for a while. it started as pure curiosity, the same "but how does it actually WORK underneath" itch that got me into engines and compilers, just pointed at systems and networks instead. tools like ossint came out of it, memsnoop later too. but there's a difference between "i mess with this for fun" and having an actual credential that says i know what i'm doing. it makes the security half of my identity feel real instead of a hobby i mention

what i like is how much it rhymes with the rest of what i do. writing a compiler taught me you have to understand a language down to the token to bend it. security is the same energy, understand the system down to the byte and the assumptions, and suddenly you see all the doors nobody meant to leave open. "software developer AND cyber-pentester" isn't two different people, it's the same curiosity aimed at build vs break. now i've got the paper to back the second half
