# made my first actual library - JSRC framework

## Date : 17-06-2022

so ive been making little löve games for a while now and i kept noticing i was rebuilding the SAME ui stuff over and over. buttons, menus, text inputs, sliders. every single new project id copy paste the same messy code and tweak it. got tired of that real quick

so i finally sat down and built JSRC-FRAMEWORK, a proper gui library for löve games. the idea is you just drop it in and you get all the ui components without rebuilding them from scratch every time. this is actually my first repo where i thought "wait, other people could use this too" instead of just building for myself

designing something for OTHER developers is a totally different headspace. like i had to think about how someone else would use my code, make the api not confusing, actually write examples. i wasnt great at the documentation part lmao but the fact that i shipped a reusable thing felt like a real step up. this is the first thing on my github and honestly a decent place to start
