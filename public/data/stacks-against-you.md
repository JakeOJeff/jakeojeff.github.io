# stacks against you - a chat app with NO database

## Date : 12-07-2025

started stacks-against-you today and im kind of obsessed with the concept. its a privacy focused chatting app with ZERO database. like none. nothing gets stored anywhere. messages exist during the session and then theyre just... gone. no logs, no history, no data sitting on a server waiting to get leaked

building something stateless like this is a really fun constraint because it forces you to rethink everything. normally chat apps are all about persistence, storing messages, syncing history across devices. i threw all of that out. if theres no database theres nothing to breach, nothing to subpoena, nothing to sell. privacy by architecture instead of privacy by policy, which honestly is the only privacy i actually trust

its built in typescript and its the kind of idea that sounds almost too simple until you try to make it actually work smoothly. this is a summer of making project and its probably my favorite thing ive built this year conceptually. the security brain and the builder brain finally working on the same project
