# fluid simulation is actually mesmerizing - spellfluid

## Date : 11-10-2025

started Spellfluid this week and i cannot stop staring at it. its a fluid simulator based on euler-grid fluids with solid objects, built in löve. basically simulating how liquids and gases actually flow, with stuff they can flow around. watching it run is genuinely hypnotic, like a lava lamp you built with math

this came from reading about fluid dynamics and going "wait i think i could actually implement this." the euler grid method treats space as a grid of cells and you track how stuff moves between them each frame, velocity, density, pressure, all of it. its heavy on the math and getting it stable is finicky, one wrong coefficient and the whole thing either freezes solid or explodes into chaos. but when it works it looks REAL, fluid curling around obstacles exactly like it should

this is easily one of my favorite technical projects, it hit 5 stars which for me is a lot lol. and it scratches the same itch as the physics engine, building a fundamental simulation of how the world works from scratch. theres something almost magical about typing out equations and watching something that looks alive come out the other end. might do a web demo of this so people can play with it without downloading anything
