# getting into security stuff - ossint

## Date : 16-06-2025

so alongside all the game dev, ive been getting deeper into the security/pentesting side of things and i started building OSSint this week. its an open source intelligence tool that actually makes sense, thats the whole tagline. the idea is gathering public info about people/things from across the web and actually presenting it in a way thats useful instead of a wall of raw data

this scratches a completely different part of my brain than löve does. game dev is about creating a world, security is about understanding systems deeply enough to see the cracks. i do pentesting on machines, web, and android, and building my own tooling for it just makes sense. plus you learn SO much about how the internet actually works when you build something that pokes at it

theres an ethics thing here im very aware of, the same tool can help someone secure themselves or invade someone elses privacy, so it matters what you build and how you frame it. i lean hard toward "understand systems to protect them." anyway this is part of the summer of making grind too, trying to actually ship consistently this year instead of starting ten things and finishing none
